## 安装相关

### 为什么在 LSPosed 中看不到 LSPilot 模块？

请确保：
- LSPosed 框架已正确安装并激活
- LSPilot APK 已正确安装
- 尝试重启设备后再检查

### 支持哪些 Android 版本？

LSPilot 支持 Android 14.0 及以上版本，架构要求为 arm64-v8a。

### 支持 x86 设备吗？

目前仅支持 arm64-v8a 架构，暂不支持 x86 设备。

### 如何验证安装是否成功？

安装完成后，打开 LSPilot 应用，如果能看到主界面且功能正常响应，说明安装成功。

### 模块激活后还是不生效？

请检查以下几点：
1. 是否已在 LSPosed 中启用模块
2. 是否已选择目标应用作为作用域
3. 是否已强制停止并重启目标应用
4. 目标应用版本是否与模块兼容

## 使用相关

### 插件脚本如何编写？

插件存储在 `Android/media/<包名>/LSPilot/Plugin/` 目录下，支持 Java、Lua、JS 三种语言。创建插件后可直接在应用内编辑运行。详细 API 文档请参考[插件开发](/plugins)章节。

### 某些功能无法使用？

部分功能依赖于目标应用的版本和系统环境，请确保：
- 模块已正确激活
- 选择了适当的作用域
- 目标应用版本兼容
- 重启目标应用使模块生效

### AI 生成的脚本不生效？

可能的原因：
- 描述需求不够清晰
- 目标应用版本与代码不匹配
- 类名或方法名被混淆

建议结合逆向分析工具先获取准确信息，再让 AI 生成脚本，并基于运行结果迭代优化。

### 如何获取最新版本？

通过以下渠道获取最新版本：
- [GitHub Releases](https://github.com/Xposed-Modules-Repo/me.yun.lspilot/releases)
- [Telegram 频道](https://t.me/LSPilot)

### 描述需求时有什么技巧？

越清晰的描述，AI 分析越准确。尽量说明：
- 目标功能（会员校验、登录、支付等）
- 期望行为（绕过、修改、追踪等）
- 已知信息（类名、方法名、字符串等）

## 反馈与支持

### 反馈渠道

如果以上解答没有解决你的问题，可以通过以下渠道获取帮助：

#### Telegram 频道

关注 [LSPilot](https://t.me/LSPilot) Telegram 频道，获取版本更新通知和使用教程。

#### GitHub

在 [GitHub Issues](https://github.com/Xposed-Modules-Repo/me.yun.lspilot/issues) 提交问题反馈或功能建议，建议包含：

- 设备型号和 Android 版本
- LSPosed 版本
- LSPilot 版本
- 目标应用及版本
- 问题描述和复现步骤
- 相关日志或截图

### 提交反馈建议

为了更快地解决问题，提交反馈时请尽量提供：

1. **环境信息**：设备型号、Android 版本、LSPosed 版本、LSPilot 版本
2. **目标应用**：应用名称、版本号、包名
3. **问题描述**：期望行为、实际行为、复现步骤
4. **相关日志**：错误信息、崩溃日志等
5. **截图或录屏**：如果方便的话
