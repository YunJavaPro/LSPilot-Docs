## 逆向分析

LSPilot 提供了强大的逆向分析工具集，帮助你在手机上完成大部分逆向工作。

| 功能 | 描述 |
| --- | --- |
| jadx 反编译 | 在手机上直接反编译 APK/DEX 文件，查看 Java 源码 |
| DexKit 搜索 | 高性能 DEX 字节码搜索，支持类/方法/字段链式查询 |
| Dex 搜索器 | 基于 smali/baksmali 的 DEX 文件分析与搜索 |
| 资源读取器 | 解析和查看 Android 资源文件 (ARSC) |
| 类型签名解析 | 解析 Java 类型签名，辅助方法查找 |
| 字符串分析器 | 分析和定位 DEX 中的字符串引用 |

### jadx 反编译

在手机上直接反编译 APK/DEX 文件，无需电脑辅助。支持查看完整的 Java 源码，方便分析应用逻辑。

### DexKit 搜索

高性能 DEX 字节码搜索引擎，支持：

- 类搜索
- 方法搜索
- 字段搜索
- 链式查询

特别适合处理混淆代码，能快速定位目标。

### 资源读取器

解析和查看 Android 资源文件 (ARSC)，方便分析应用资源结构。

## 插件引擎

LSPilot 内置强大的脚本引擎，支持多种脚本语言。

| 功能 | 描述 |
| --- | --- |
| BeanShell (Java) | 兼容 Java 语法，可直接调用 Android API |
| Lua | 轻量级 LuaJIT 引擎，适合快速编写调试脚本 |
| JavaScript | Rhino 引擎，支持现代 JS 语法 |

### BeanShell (Java)

兼容 Java 语法，可以直接调用 Android API。适合熟悉 Java 的开发者，能充分发挥 Xposed 的全部能力。

### Lua

基于 LuaJIT 引擎，轻量高效。适合快速编写调试脚本，语法简洁。

### JavaScript

基于 Rhino 引擎，支持现代 JS 语法。适合前端开发者使用。

## AI 驱动分析

LSPilot 的核心特色是 AI 驱动的逆向分析：

- **自然语言交互**：用大白话描述需求
- **自动定位**：AI 自动查找目标类和方法
- **代码生成**：自动生成 Hook 脚本
- **思路讲解**：提供清晰的逆向思路说明

## 动态注入

通过 LSPosed 框架实现：

- 一键动态注入脚本
- 实时验证结果
- 无需反复编译模块
- 运行时下发修复逻辑
